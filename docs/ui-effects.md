# UI-эффекты

## Где смотреть стили

Основные визуальные эффекты находятся в файлах:

- `assets/css/styles.css`
- `assets/css/seo.css`
- `assets/css/accessibility.css`

## Что проверять

- Hover не должен быть единственным способом понять действие.
- Фокус с клавиатуры должен быть заметен.
- Карточки не должны прыгать слишком сильно.
- CTA-кнопки должны оставаться читаемыми на светлом и темном фоне.
- На мобильном экране эффекты не должны мешать нажатию.

## Базовое правило

Визуальные эффекты должны помогать восприятию сайта, а не отвлекать от обращения к ипотечному брокеру.
