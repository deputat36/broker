# Заметки по доступности сайта

Документ фиксирует уже реализованные элементы доступности и то, что нужно проверять после крупных правок.

## Что уже есть

- Skip-link для быстрого перехода к основному содержимому.
- Видимые состояния `focus-visible` для ссылок, кнопок, полей ввода и раскрывающихся элементов.
- Дополнительная подсветка карточек через `focus-within`.
- ARIA-состояния мобильного меню.
- Закрытие мобильного меню по `Escape`.
- `aria-live` для результата ипотечного калькулятора.
- `aria-live` для кнопок копирования номера MAX.
- Поддержка `prefers-reduced-motion`.
- Поддержка `prefers-contrast: more`.
- Печатные стили через `assets/css/print.css`.

## Что проверять после правок

- Меню открывается и закрывается с клавиатуры.
- Фокус виден на кнопках и ссылках.
- Skip-link появляется при переходе клавишей Tab.
- Калькулятор сообщает ошибки и результат без перезагрузки страницы.
- Кнопка MAX сообщает результат копирования.
- На мобильном экране sticky-панель не перекрывает важный контент.
- В режиме уменьшения движения нет заметных анимационных эффектов.
- В режиме повышенной контрастности ссылки, карточки и кнопки остаются читаемыми.

## Что не делать

- Не убирать `aria-label` у меню и кнопок связи.
- Не отключать outline без равноценной замены.
- Не делать CTA только цветом без текстового смысла.
- Не использовать изображения без осмысленного alt, если изображение несет смысл.
