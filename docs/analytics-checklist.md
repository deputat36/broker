# Чек-лист аналитики

## Что уже подготовлено

В проекте есть условное подключение Яндекс Метрики через `_config.yml`.

Пока значение пустое:

```yaml
yandex_metrika_id: ""
```

код счетчика не выводится.

## Как включить Метрику

1. Создать счетчик в Яндекс Метрике.
2. Взять ID счетчика.
3. Вставить ID в `_config.yml`.
4. Дождаться деплоя GitHub Pages.
5. Проверить исходный код страницы.

Пример:

```yaml
yandex_metrika_id: "12345678"
```

## Цели, которые нужно создать

В Метрике нужно создать JavaScript-цели:

- `phone_click`;
- `vk_click`;
- `max_copy`;
- `consultation_click`;
- `contacts_click`;
- `calculator_input`.

## Проверка после подключения

После деплоя проверить:

- клик по телефону;
- переход во ВКонтакте;
- копирование номера для MAX;
- переход на консультацию;
- переход на контакты;
- ввод данных в ипотечном калькуляторе.

## Важно

Без согласования не менять ID счетчика, телефон, ссылку ВКонтакте и логику обработки персональных данных.
