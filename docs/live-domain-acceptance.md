# Приемка боевого домена sterlikova-ipoteka.ru

Документ используется после изменения DNS, GitHub Pages, CNAME, sitemap, robots или основных SEO-настроек.

## 1. GitHub Pages

Проверить в `Settings → Pages`:

- источник публикации — GitHub Actions;
- custom domain — `sterlikova-ipoteka.ru`;
- DNS check завершена успешно;
- включен `Enforce HTTPS`;
- в репозитории существует файл `CNAME` со значением `sterlikova-ipoteka.ru`.

## 2. DNS

У регистратора домена проверить:

- корневой домен указывает на GitHub Pages;
- нет конфликтующих старых A, AAAA или CNAME-записей;
- `www` либо корректно перенаправляется, либо настроен как CNAME на GitHub Pages;
- после изменений закончился период обновления DNS.

## 3. Основные адреса

Открыть в обычном и приватном окне браузера:

- `https://sterlikova-ipoteka.ru/`;
- `https://sterlikova-ipoteka.ru/uslugi/`;
- `https://sterlikova-ipoteka.ru/konsultaciya/`;
- `https://sterlikova-ipoteka.ru/kalkulyator-ipoteki/`;
- `https://sterlikova-ipoteka.ru/polezno/`;
- `https://sterlikova-ipoteka.ru/kontakty/`;
- `https://sterlikova-ipoteka.ru/robots.txt`;
- `https://sterlikova-ipoteka.ru/sitemap.xml`;
- `https://sterlikova-ipoteka.ru/site.webmanifest`;
- `https://sterlikova-ipoteka.ru/assets/img/tatyana-social.jpg`.

Для каждого URL проверить:

- нет ошибки сертификата;
- нет бесконечного редиректа;
- адрес остается на основном домене;
- страница не открывается как 404;
- кириллица отображается корректно.

## 4. Главная страница

В исходном коде главной проверить:

- canonical: `https://sterlikova-ipoteka.ru/`;
- `og:url` на основном домене;
- `og:image` открывается;
- Schema.org не содержит `deputat36.github.io/broker`;
- телефон указан как `+79030250807`;
- нет неподтвержденного обещания `15 мин`;
- нет формулировок гарантированного одобрения.

## 5. Robots и sitemap

В `robots.txt` должна быть строка:

```text
Sitemap: https://sterlikova-ipoteka.ru/sitemap.xml
```

В sitemap проверить:

- XML открывается без ошибки;
- URL используют HTTPS и основной домен;
- отсутствует `/404.html`;
- присутствуют главная, услуги, гео-страницы и полезные материалы;
- нет технического адреса GitHub Pages.

## 6. Мобильная приемка

На телефоне проверить:

- меню открывается и закрывается;
- первый экран не выходит за границы;
- фото отображается;
- фиксированная панель не перекрывает контент;
- телефон кликается;
- MAX копирует номер;
- ВКонтакте открывается;
- калькулятор пересчитывает платеж;
- формы ввода не вызывают горизонтальную прокрутку.

## 7. Яндекс Вебмастер

После доступности домена:

- добавить `https://sterlikova-ipoteka.ru/`;
- подтвердить права;
- отправить `/sitemap.xml`;
- проверить robots;
- проверить главную через инструмент переобхода;
- отслеживать исключенные страницы и дубли canonical.

## 8. Google Search Console

После доступности домена:

- добавить ресурс домена или URL-prefix;
- подтвердить права;
- отправить `/sitemap.xml`;
- проверить главную через URL Inspection;
- запросить индексирование основных страниц;
- отслеживать ошибки Pages и Core Web Vitals.

## 9. Фиксация результата

После проверки записать:

- дата и время приемки;
- кто проверил;
- DNS check: успешно / ошибка;
- HTTPS: включен / не включен;
- главная: работает / ошибка;
- robots: работает / ошибка;
- sitemap: работает / ошибка;
- Яндекс Вебмастер: подключен / не подключен;
- Search Console: подключена / не подключена;
- найденные проблемы и ссылки на GitHub issues.

## 10. Критерий готовности к рекламе

Сайт готов к активной рекламе только если:

- основной домен стабильно открывается по HTTPS;
- GitHub Actions завершает сборку без ошибки;
- главная, контакты и консультация работают;
- robots и sitemap доступны;
- подключена аналитика;
- телефон, MAX и ВКонтакте проверены на мобильном устройстве;
- фото Татьяны отображается корректно;
- нет обещаний гарантированного одобрения.
