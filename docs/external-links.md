# Внешние ссылки

Документ фиксирует правило работы с внешними ссылками на сайте.

## Основной принцип

Все внешние ссылки должны быть безопасными и понятными для пользователя.

## Что уже реализовано

В `assets/js/main.js` есть функция `enhanceExternalLinks()`.

Она находит внешние ссылки и автоматически добавляет к ним:

```html
rel="noopener noreferrer"
```

Это снижает риск небезопасного взаимодействия с внешними страницами и делает поведение ссылок единообразным.

- Внешние ссылки не должны вести на непроверенные страницы.
