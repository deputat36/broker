# Прозрачность ставки ипотечного калькулятора

## Цель

Калькулятор помогает сравнивать пользовательские сценарии, но не должен выглядеть как источник актуальной ставки банка.

Ранее обе формы содержали предзаполненное значение `18`. Даже при пояснении о предварительном расчёте такое значение могло восприниматься как текущая рыночная ставка или предложение конкретного банка.

## Пользовательский контракт

На главной и странице `/kalkulyator-ipoteki/`:

- поле называется «Предполагаемая ставка, % годовых»;
- числовое значение по умолчанию отсутствует;
- placeholder не содержит примерной цифры;
- пользователь указывает ставку из предложения банка либо задаёт собственный сценарий;
- рядом с полем явно указано, что калькулятор не подставляет актуальную ставку банка;
- пустое поле считается незаполненным, а не ставкой 0%;
- до ввода корректной ставки кнопка переноса расчёта в заявку скрыта.

## Почему ставка не подставляется автоматически

Банковские ставки быстро меняются и зависят от программы, объекта, первоначального взноса, страхования, параметров клиента и дополнительных условий. Статичное значение без подтверждённого источника и даты проверки быстро становится недостоверным.

Сайт не использует неподтверждённую ставку для привлечения заявки и не обещает условия банка.

## Автоматическая защита

`scripts/audit-calculator-rate-transparency.py` проверяет исходные и собранные страницы:

- калькулятор присутствует только на главной и `/kalkulyator-ipoteki/`;
- каждая форма содержит ровно одно поле `rate`;
- у поля нет непустого `value`;
- placeholder равен «Введите ставку» и не содержит цифр;
- label использует формулировку «Предполагаемая ставка, % годовых»;
- `aria-describedby` ведёт к пояснению о пользовательском сценарии и отсутствии актуальной ставки банка.

Поведенческий тест `scripts/test-mortgage-calculator-application-action.js` дополнительно подтверждает, что пустая ставка:

- не превращается в 0%;
- показывает понятную ошибку;
- получает `aria-invalid="true"`;
- скрывает кнопку переноса расчёта в онлайн-заявку.
