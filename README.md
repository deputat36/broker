# Сайт ипотечного брокера Татьяны Стерликовой

SEO-сайт для сбора обращений по ипотечным консультациям и сопровождению в Борисоглебске, Борисоглебском городском округе, Грибановском и Поворинском районах.

Рабочий репозиторий: `deputat36/broker`.

Публичный адрес GitHub Pages:

`https://deputat36.github.io/broker/`

## Главная задача сайта

Сайт должен приводить клиентов из поиска и рекламы, объяснять услуги простым языком и переводить посетителя в обращение:

- звонок по телефону `8 903 025-08-07`;
- копирование телефона для MAX;
- переход на страницу ВКонтакте `https://vk.com/tatyanasterlikova`.

Формы заявок пока не используются, чтобы не усложнять обработку персональных данных и инфраструктуру.

## Публикация на GitHub Pages

Рекомендуемый вариант:

1. Открыть `Settings → Pages`.
2. В качестве source выбрать `GitHub Actions`.
3. Запустить workflow `Deploy Jekyll site to Pages` вручную или дождаться запуска после push в `main`.

Запасной вариант:

1. `Settings → Pages`.
2. Source: `Deploy from a branch`.
3. Branch: `main`.
4. Folder: `/root`.

## Работа с доменом

Рекомендуемый домен:

`sterlikova-ipoteka.ru`

После покупки домена нужно:

1. Добавить файл `CNAME` со строкой домена.
2. В DNS домена прописать записи для GitHub Pages.
3. В `_config.yml` заменить:
   - `url` на основной домен;
   - `baseurl` на пустое значение `""`.
4. Проверить canonical, Open Graph, `robots.txt` и `sitemap.xml`.
5. Включить HTTPS в настройках GitHub Pages.
6. Отправить sitemap в Яндекс Вебмастер и Google Search Console.
