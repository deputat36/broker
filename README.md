# broker

SEO-сайт ипотечного брокера Татьяны Стерликовой для GitHub Pages.

## Что уже сделано

- Главная страница с продающим оффером и CTA.
- Отдельные SEO-страницы услуг:
  - подбор ипотеки;
  - семейная ипотека;
  - ипотека с материнским капиталом;
  - сложная ипотечная ситуация.
- Отдельные локальные страницы:
  - Борисоглебск;
  - Грибановский район;
  - Поворино и Поворинский район.
- FAQ с микроразметкой FAQPage.
- Контакты с телефоном, MAX через телефон и ссылкой ВКонтакте.
- Jekyll-шаблон `_layouts/default.html`, стили, `robots.txt`, `sitemap.xml`, `404.html`.

## Публикация на GitHub Pages

В настройках репозитория включить Pages:

- Source: Deploy from a branch.
- Branch: `main`.
- Folder: `/root`.

После публикации сайт будет доступен по адресу:

`https://deputat36.github.io/broker/`

## Домен

Рекомендуемый домен:

`sterlikova-ipoteka.ru`

После покупки домена нужно:

1. Добавить файл `CNAME` со строкой `sterlikova-ipoteka.ru`.
2. В настройках DNS домена прописать записи для GitHub Pages.
3. В `_config.yml` заменить `url` и `baseurl` под основной домен.
4. Проверить canonical, `robots.txt` и `sitemap.xml` после сборки.
5. Включить HTTPS в настройках GitHub Pages.
